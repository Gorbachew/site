Для того чтобы добавить файлы в github необходимо:
git init - указать папку репозитории
git add * (если выделить все файлы) или . (если только изменненные новые)
git commit -m "v2" - сконвертировать файлы в новую версию
git remote add origin https://github.com/Gorbachew/site.git - выбрать репозиторий github
git push -u origin master - отправить файлы в github

Для того, чтобы получить файлы из репозитории:
git clone https://github.com/Gorbachew/site.git - в папке куда установить файлы

Установка виртуального окружения:
virtualenv --python=python3.6 myvenv
source myvenv/bin/activate
